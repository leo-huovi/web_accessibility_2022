Layout

Website Accessibility I - Tuni 2022

⚠️ Attention! ⚠️

Layout can emphasize content through visual and non-visual methods. For example, colors, shapes, symbols, positioning.

Additionally

Layout can guide the user's eye towards different things on the page. However, for all kinds of layout ideas, non-visual assistive techonlogies must be considered.
